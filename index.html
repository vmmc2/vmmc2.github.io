  • NEMESIS

    • Nemesis is a cache simulator of 1024 words made using Python. It can simulate 16 different types of cache and supports both write access and read access. It was mainly developed using the RISC-V architecture as a basis and inspiration.

    Relevant technologies used: Python.

  • RISC-V DISASSEMBLER AND ISA SIMULATOR

    • This project consists in a disassembler made for the RISC-V architecture. It was developed to work only in the RV-32I extension. It receives a set of instructions in hexadecimal format, it executes these instructions (changing both the content of registers and memory). It also generates the RISC-V assembly code equivalent to the instructions in the hexadecimal that were passed as input, and puts this assembly code into a .txt file.

    This project was conceived using C/C++ and it was mainly based on a Assignment passed to the students from UC Berkeley.

  • HANGMAN BOOTLOADER
